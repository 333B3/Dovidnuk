Ця програма призначена для збереження та організації контактної інформації про людей, з якими ви спілкуєтеся або взаємодієте. Основна мета такої програми - надати вам зручний спосіб зберігання, пошуку і управління вашими контактами. Більш обширну інструкцію ви можете переглянути в пояснювальній записці до курсової роботи.
